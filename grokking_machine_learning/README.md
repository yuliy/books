# Глава 1. Что такое машинное обучение? Это здравый смысл, проявляемый компьютером
Какое же дебильное определение...

> **Искусственный интеллект** - набор всех задач, в которых компьютер может принимать решения.

> **Машинное обучение** - набор всех задач, в которых компьютер может принимать решения на *основе данных*. МО является частью ИИ.

> **Глубокое обучение** - часть машинного обучения, использующая объекты, называемые *нейронными сетями*.
