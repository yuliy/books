# Глава 1. Что такое машинное обучение? Это здравый смысл, проявляемый компьютером
Какое же дебильное определение...

> **Искусственный интеллект** - набор всех задач, в которых компьютер может принимать решения.

> **Машинное обучение** - набор всех задач, в которых компьютер может принимать решения на *основе данных*. МО является частью ИИ.

> **Глубокое обучение** - часть машинного обучения, использующая объекты, называемые *нейронными сетями*.

![](./img/pic_1_3.png)


Автор, анализируя процесс, с помощью которого люди принимают решения, основанные на опыте, выделяет концепцию **вспоминания - формулирования - прогнозирования**.

![](./img/pic_1_4.png)

> **Модель** - набор правил, которые представляют наши данные и могут быть задействованы для прогнозирования.

> **Алгоритм** - процедура или набор шагов, используемых для решения задачи или выполнения вычисления. Здесь цель алгоритма - построения модели.

> **Признак (feature)** - любой свойство или характеристика данных, которые модель может применять для прогнозирования.
